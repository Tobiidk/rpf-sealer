Prior-art acknowledgement
-------------------------
RpfSealer was prompted by the 2016-era community tool "ArchiveFix" (author
unknown), which popularised the drag-and-drop workflow for NG-sealing GTA V
RPF archives. No ArchiveFix source or binary content is redistributed here.
RpfSealer is an independent implementation of equivalent functionality,
modernised for GTA V Enhanced and other current builds, with the
key-derivation path replaced entirely by CodeWalker's magic-blob approach
(which supersedes the hash-window-scan pipeline ArchiveFix used).

Non-redistribution of Rockstar Games IP
---------------------------------------
This project does not redistribute any Rockstar Games intellectual property.
The magic blob contains only derived cryptographic constants (permutation
tables and key schedules) that are themselves only unlocked by an AES key
extracted at runtime from the user's own installed copy of GTA V. No game
code, assets, or materials are bundled or replayed.
